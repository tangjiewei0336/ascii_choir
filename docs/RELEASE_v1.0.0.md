# ASCII Choir v1.0.0

用 ASCII 写简谱，即可播放音乐。支持多种乐器、语音朗读与 VOICEVOX 歌声合成。

---

## 下载与安装

1. 根据系统下载对应文件：
   - **macOS**：`ASCII_Choir-macOS.zip`
   - **Windows**：`ASCII_Choir-Windows.zip`
2. 解压 zip 文件
3. 进入解压后的 `ASCII_Choir` 文件夹，双击运行：
   - macOS：`ASCII_Choir`
   - Windows：`ASCII_Choir.exe`

无需安装，解压即用。

---

## 基本使用

### 1. 打开程序

运行后，主界面分为：
- **左侧**：工作区文件列表（可打开已有简谱或新建）
- **中间**：编辑区（输入简谱）
- **上方**：播放、停止等按钮

### 2. 写一段简谱

在编辑区输入，例如：

```text
\tonality{0}
\beat{4/4}
\bpm{120}

|1 2 3 4|5 6 7 1|
```

- `\tonality{0}`：调性（0=C 调）
- `\beat{4/4}`：拍号
- `\bpm{120}`：速度
- `|1 2 3 4|`：小节，数字 1–7 表示音符 do–si

### 3. 播放

点击 **播放** 按钮，即可听到钢琴演奏。

### 4. 换乐器

在旋律行首加 `[乐器名]`，例如：

```text
& [guitar]|1 2 3 4|5 6 7 1|
& [cello]|1 2 3 4|5 6 7 1|
```

常用乐器：`[piano]`、`[guitar]`、`[cello]`、`[drums]` 等。

## 常用功能

### 保存与导出

- **文件 → 保存**：保存为 `.choir` 文件
- **文件 → 导出 MP3**：导出为音频（需安装 ffmpeg）

### 添加歌词与歌声

1. 打开 **TTS 菜单 → VOICEVOX 音色选择**
2. 选择音色，点击试听
3. 复制「歌词命令」到剪贴板，粘贴到简谱中
4. 播放时，该篇章会用人声歌唱

**使用歌声需满足其一**：
- 程序已内置本地库（开箱即用，首次合成需要下载）
- 或运行 Docker：`docker run -p 50021:50021 voicevox/voicevox_engine:cpu-latest`

### 篇章间插入语音

在篇章之间（双换行分隔）加入：

```text
\tts{这里是旁白}{zh}
```

支持中文、日语、英语。

### 多声部

用 `&` 表示一条旋律，多行 `&` 同时演奏：

```text
& |1 2 3 4|
& |5 6 7 1|
```

## 内置工作区

程序自带多个工作区，可从左侧文件列表打开，作为学习与参考。

| 工作区 | 说明 |
|--------|------|
| **示例** | 各类功能示例：单声部、多声部、变调、和声、鼓、歌声合成、小星星等 |
| **大阪环状线** | 大阪环状线、阪和线、京都等车站的接近/通过/到着发车音乐 |
| **山手线** | 山手线车站音乐（关门、大崎等） |
| **lib** | 共享和弦库，供 `\import{chords.choir}` 引用，可搭配其他工作区使用 |

建议先打开 **示例** 中的 `小星星.choir`、`单声部.choir` 体验播放，再尝试 `歌声合成.choir` 感受人声合成。
